# TopDownShooter
A 3D top-down shooter game prototype that developed with Unity3D. </br>

## Goal
- Kill the incoming enemies, avoid from enemies' self-detonation and stay alive as long as you can. </br>

## Power-Ups
- Max Ammo </br>
